WildFly Support

A new WildFly server adapter was added.

A new WildFly server adapter has been added. It is almost entirely equivilent to the previous JBoss AS 7.x server adapters, and you won't notice many changes when using it to create, start, stop, or publish to your WildFly Application Server, but this is just another step forward for support of the upcoming WildFly server by JBoss.
